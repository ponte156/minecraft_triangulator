# 마인크래프트 삼각 측량
### 프로젝트 소개
> 이 프로젝트는 마인크래프트 플레이시 중 엔더 유적을 찾을 때 사용되는 **엔더의 눈 소비를 최소화**하는 프로그램입니다.
>**삼각 측량 알고리즘**을 이용해서 목표 위치와의 거리를 쉽게 측정할 수 있습니다.
### 💾 다운로드
최신 버전파일은 우측 **Releases** 섹션이나 아래 링크를 통해 다운로드 할 수 있습니다.

### 사용법
1. 현재 바라보는 **방위**를 정한다.
2. **지점1**에서 바라본 목표와의 각도를 기입한다.
3. **지점2**에서 바라본 목표와의 각도를 기입한다.
4. **지점2**와 **지점1** 간의 거리를 **지점간 거리**에 기입한다.
5. **계산** 버튼을 클릭하면 삼각 측량 값이 출력된다.
